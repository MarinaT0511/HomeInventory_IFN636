### Home Inventory Management ###
This application is designed to assist users in managing their personal belongings and tracking detailed information digitally. The application allows users to create, update and manage their home inventory, as well as view details about each item. 

### Prerequisites ###
react
Node.js
MongoDB
Mocha
Chai
bootstrap
react-bootstrap

### Installation ###
1, Clone repository
2, Run "npm install" in both backend and frontend directories
3, Run "node server.js" in the backend directory
4, Run "npm start" in the frontend directory
